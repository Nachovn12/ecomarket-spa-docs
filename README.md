# EcoMarket SPA - Proyecto de Microservicios

Proyecto académico de Desarrollo Fullstack I orientado a la migración de EcoMarket SPA desde un sistema monolítico hacia una arquitectura basada en microservicios.

## Microservicios definidos

- ms-usuarios-identidad
- ms-catalogo
- ms-inventario-abastecimiento
- ms-pedidos-ventas
- ms-logistica-envios
- ms-administracion-soporte
- ms-reportes

## Tecnologías consideradas

- Java
- Spring Boot
- Maven
- API REST
- JPA / Hibernate
- Postman
- JUnit
- HATEOAS
- Git / GitHub

## Flujo Git

- main: rama estable/final del proyecto.
- develop: rama de integración del equipo.
- feature/ms-*: ramas por microservicio.

El equipo trabajará sobre develop y ramas feature por microservicio. No se trabajará directamente sobre main.
