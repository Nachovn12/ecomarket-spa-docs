## Responsable

## Objetivo

Reunir evidencias técnicas para la entrega y defensa final del proyecto backend basado en microservicios.

Estas evidencias respaldan el trabajo realizado en Jira, GitHub, Postman, JUnit y la implementación técnica de los microservicios desarrollados durante el proyecto EcoMarket SPA.

---

## Evidencias incluidas

1. Jira: épicas, historias, tareas y sprints.
2. GitHub: ramas, commits y pull requests.
3. Postman: endpoints REST validados.
4. JUnit: pruebas unitarias y pruebas de contexto.
5. REST, CRUD y HATEOAS.
6. Arquitectura de microservicios.
7. Preparación de defensa técnica individual.

---

## Relación con la rúbrica

Las evidencias respaldan los siguientes criterios técnicos:

- Arquitectura CSR: Controller, Service, Repository y Model.
- Persistencia con JPA/Hibernate.
- CRUD conectado a base de datos.
- API REST con métodos HTTP correctos.
- Validaciones con Bean Validation.
- Manejo centralizado de errores.
- Uso de HATEOAS.
- Pruebas con JUnit.
- Uso de GitHub mediante ramas, commits y Pull Requests.
- Organización del trabajo mediante Jira.
- Preparación para defensa técnica individual.
